# Alfred 功能对标路线图

逐项 UI、快捷键、预期行为、当前差距和验收条件请以 `ALFRED_PARITY_SPEC.md` 为准；本文件仅保留高层实施路线。

目标是实现 Alfred 的主要个人效率能力，同时坚持原生、离线优先、无市场、无遥测和最小权限。这里的“对标”指功能与交互思路，不复制 Alfred 的代码、资源、名称或界面细节。

## 已实现

| 能力 | 当前实现 |
| --- | --- |
| 应用启动 | 扫描系统与用户 Applications 目录，支持名称、拼音全拼/首字母匹配和目录变更刷新 |
| 默认结果 | 应用、计算器、词典和少量 Spotlight 文件统一排序 |
| 学习排序 | 根据使用次数和最近使用时间加权，只持久化 SHA-256 标识哈希 |
| 文件搜索 | `open/打开` 搜索文件名，`find/查找` 在访达定位，`in/内容` 搜索索引内容 |
| 文件导航 | 输入 `/` 或 `~` 浏览目录，目录优先，回车进入下一级 |
| 文件动作 | 类型化动作面板、Quick Look、访达定位、复制路径、文件缓冲与批量动作 |
| Quick Look | `Command + Y` 在主窗口内显示系统原生预览 |
| 计算器 | 安全表达式、白名单函数/常量、连续计算和离线单位换算 |
| 词典与拼写 | macOS Dictionary Services 释义和 NSSpellChecker 建议 |
| 剪贴板历史 | 文本/文件/可选图片、类型筛选、固定、忽略应用、暂停与加密保留 |
| Snippets | 加密片段、关键词搜索、剪贴板保存和动态占位符 |
| 最近文档 | 加密记录经 YTools 打开的文件，支持关键词检索 |
| Large Type | Command+L 或文本动作打开原生大字窗口 |
| 安全系统动作 | 可配置关键词；屏保、显示/确认后由 Finder 清空废纸篓、关闭显示器、勿扰与外观设置入口；应用隐藏/正常退出 |
| 全局快捷键 | 默认 `Option + Space` 启动器、`Option + Command + C` 剪贴板，可在设置中录制修改 |
| 设置与外观 | 原生侧边栏设置页、极简/经典/现代/玻璃风格、深浅色、结果密度、副标题、记忆窗口位置和可隐藏菜单栏入口 |
| 默认输入语言 | 可跟随系统当前输入源，或在每次显示启动器/剪贴板时切换到指定本机输入源 |
| Finder 标签 | `tag/标签` 通过 Spotlight 搜索本机 Finder 标签 |

## 第二阶段：日常高频能力（已完成主路径）

- 已完成文件缓冲、最近文档、Snippets 主路径、搜索范围、忽略应用、登录启动和默认关闭的图片历史。
- 已完成独立系统命令设置页、可配置关键词、屏保、显示废纸篓、确认后清空废纸篓和显示器睡眠；勿扰与外观只打开系统设置页。音量、静音、锁屏等继续遵守“原生可用且不新增宽权限”原则逐项评估。
- 尚待增强：Snippets 独立浏览/导入导出和可选自动粘贴；批量失败报告、剪贴板磁盘占用及单条文本长度限制已完成。

## 2026-07 Alfred 官方功能复核

| 分类 | 能力 | 决策 |
| --- | --- | --- |
| 已补入 | 剪贴板单条文本长度上限、Force Keyboard、Finder 标签搜索 | 原生、本地、无需新增宽权限，已实现 |
| 已具备 | 应用/文件搜索、文件导航与缓冲、Universal Actions、Quick Look、计算器、词典、拼写、剪贴板、Snippets、Large Type、学习排序 | 继续按 YTools 的受控动作与加密存储实现，不复制 Alfred 资源 |
| 下一批本地增强 | 可配置文件关键词、更多导航排序、独立 Snippets 浏览、查询历史开关、应用最近文档动作、音量/静音/锁屏等安全系统命令 | 不需要联网，但需要分别设计隐私、确认和回退行为 |
| 必须显式授权且默认关闭 | 自动粘贴、Cmd+C+C 合并、全局选中文本/文件动作、Snippet 自动展开、联系人、日历、音乐 | 涉及辅助功能、自动化或个人数据权限；启用前必须解释用途 |
| 明确排除 | Web Search、浏览器书签、1Password、在线 Workflow/插件市场、Remote、云同步、账户、遥测和任意 Shell | 与个人本机、无外链和最小权限目标冲突 |

“Alfred 有”不等于默认开启。涉及全局键盘监听、读取其他应用选择或个人数据库的功能，必须采用独立开关、最小权限和拒绝授权后完全可用的退化路径。

## 第三阶段：原生信息与高级效率

- 联系人搜索：Contacts Framework，只在用户授权后启用。
- 日历与提醒事项：EventKit，只读搜索默认启用，写入动作单独确认。
- 高级计算：常用函数/常量与离线单位换算已实现；后续补日期运算和更完整本地化数字格式。
- Large Type：主路径已实现；后续补剪贴板入口和多屏策略。
- Finder Selection：对访达当前选择执行统一动作。
- 剪贴板合并和可选的自动粘贴；自动粘贴需要辅助功能权限，默认关闭。
- 菜单栏暂停剪贴板记录已实现；后续补更完整隐私状态提示。

## 用安全的内部工具替代 Workflows

- 内置与个人扩展统一使用源码级异步 `YToolsModule`，重新编译后随应用签名。
- 每个模块声明能力并经过宿主结果策略校验；个人模块默认不获得系统控制权限。
- 模块只返回结构化结果和有限动作，不获得通用 Shell、任意 IPC 或网页容器。
- 需要网络的个人工具放入独立 XPC Service，采用域名白名单、请求超时和响应大小限制。
- 不提供在线安装、市场、远程脚本和动态加载未签名代码。

## 默认排除

- 公共插件/Workflow 市场和兼容 `.alfredworkflow`。
- Web Search、浏览器历史和书签搜索。
- 1Password 等第三方账户集成。
- 云同步、账户、遥测、心跳、广告和启动联网。
- 任意 Shell 命令输入框、HTTP/MCP Server 和远程控制接口。
